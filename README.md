# 📌 Overview

**CarbonCut** helps individuals estimate, track, and reduce their carbon footprint based on transportation habits, energy consumption, and dietary choices — making environmental awareness simple and actionable.

---

# 🚀 Features

| Feature | Description |
|---|---|
| 🌍 Footprint Estimation | Score based on transport, AC usage & diet |
| 📊 Eco Rating | Eco Friendly / Moderate Impact / High Impact |
| 🤖 Recommendations | Personalized tips to reduce emissions |
| 🎯 Top Contributor Analysis | Identifies your biggest emission source |
| 📈 Trend Dashboard | Chart.js-powered historical score tracking |
| 🎯 Goal Tracking | Set a target score and monitor progress |
| 💾 Local Storage | Persistent history, no database needed |

---

# 📌 Assumptions Made

- Carbon impact values are simplified estimates for awareness purposes, not official accounting.
- Transportation, AC usage, and diet are treated as the primary emission contributors.
- Recommendations use rule-based logic, not real-time environmental datasets.
- Users are assumed to provide accurate lifestyle inputs.

---

# 🧠 Approach & Logic

CarbonCut uses a **rule-based decision-making approach**. Each activity (transport, AC usage, diet) is assigned an environmental impact score. These are combined to compute a carbon footprint score, which then drives:

- **Eco Rating** classification
- **Top Contributor** identification
- **Personalized Recommendation** generation

The engine adapts suggestions per user input, ensuring every user receives relevant, unique advice.

---

# ⚙️ How It Works

1. **Input** — Enter transport method, travel distance, AC hours, and diet
2. **Calculate** — Rule-based logic generates a carbon footprint score
3. **Assess** — Categorized into Eco Friendly, Moderate, or High Impact
4. **Recommend** — Personalized tips based on your biggest contributor
5. **Track** — Score saved locally; visualized in history & trend charts
